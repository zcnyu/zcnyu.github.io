# カスタム6面サイコロ（Web）

6つの面に任意の文字（最大4文字）を設定して、ブラウザ上でサイコロを「ふる」ことができるシンプルなWebアプリです。入力は端末のローカルストレージに自動保存されます。v2 では CSS 3D による立体サイコロ表示と回転アニメーションを追加しました。

## 使い方
- `index.html` をブラウザで開きます。
- 「面の設定」で6つの入力欄に好きな文字を入れます（未入力は 1〜6 を使用）。
- 「サイコロをふる」をクリックすると3D回転後、前面に出た面が結果です（スクリーンリーダーへも読み上げ通知）。
- 面のラベルは1行センタリングで自動的に文字サイズが調整され、横幅のおよそ80%に収まるよう拡大・縮小します。
- 回転中および初期状態は文字を非表示にし、回転が止まって結果が確定した面だけ黒色で表示します。
- サイコロは白ベース。回転時間はおよそ2秒（`prefers-reduced-motion` が有効な場合は短縮）。
- 「デフォルトに戻す」で 1〜6 の標準面に戻せます。

## 開発/ローカルサーブ
静的ファイルのみで動作します。任意ですが、ローカルHTTPサーバで配信すると動作確認がしやすいです。

```bash
# このディレクトリで
python -m http.server 8080
# ブラウザで http://localhost:8080/ を開く
```

## 構成
- `index.html` UI本体（3DキューブのDOMを配置）
- `styles.css` 見た目（ライト/ダーク対応、3D配置とトランジション）
- `script.js` ロジック（保存・3D回転・結果通知）

## メモ
- 外部APIやネットワークは使用しません。
- 永続化は `localStorage`（キー: `dice_web_faces_v1`）。
- OSで「視覚効果を減らす」が有効な場合は回転を簡略化します（`prefers-reduced-motion`）。

```text
src/dice_web/
├── index.html

```
